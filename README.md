# Awesome OSS Companies

A curated list of commercial open-source companies, inspired by [Awesome-Selfhosted](https://github.com/awesome-selfhosted/awesome-selfhosted). This repository serves as a directory of companies that have built their businesses around open-source software, providing a valuable resource for developers, investors, and anyone interested in the commercial open-source ecosystem.

The information in this repository is sourced from [Cossmology](https://cossmology.com), a platform for tracking and analyzing the commercial open-source landscape.
